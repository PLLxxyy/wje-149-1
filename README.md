# 驾校练车预约系统

全栈驾校练车预约系统，支持学员预约练车、教练排班管理、学时记录、评分评价、管理员数据统计等功能。

## 技术栈

- **前端**：Vite + React 18 + TypeScript，端口 5173
- **后端**：Express + TypeScript + better-sqlite3，端口 3000
- **认证**：JWT + bcryptjs
- **启动**：concurrently 前后端同启
- **数据库**：SQLite（better-sqlite3），启动时自动建表和 seed

## 启动方式

```bash
# 安装所有依赖（根目录 + server + client）
npm run install:all

# 同时启动前后端
npm run dev
```

启动后访问 http://localhost:5173

## 功能说明

### 学员端
- **登录注册**：学员可注册新账号并登录
- **教练列表**：浏览所有教练，查看姓名、驾龄、评分、可约时段
- **教练详情**：查看教练详情，在日历上选择日期和时段预约练车
- **我的课程**：查看所有预约记录，支持签到、取消、评价教练
- **个人中心**：查看学时统计（已练/剩余）、历次教练评价

### 教练端
- **排班管理**：按周设置每天各时段是否可约，学员可见
- **预约管理**：查看学员预约，支持确认签到、取消、评价学员
- **我的学员**：查看学员列表、学时进度、为学员写表现备注

### 管理员端
- **数据总览**：教练/学员总数、总预约数、预约率、近7日趋势
- **教练管理**：查看所有教练详细信息和带教数据
- **学员管理**：查看所有学员学时和课程完成情况
- **教练排行**：按带教人数排行，展示评分和完成课时

### 核心流程
1. 教练设置可约时段（排班管理）
2. 学员浏览教练列表，选择教练和时段预约
3. 练车当天，学员和教练分别签到
4. 双方签到后，学时自动加一
5. 完成后双方互评（学员评教练、教练评学员）

## 注意事项

- 数据库文件 `server/driving_school.db` 启动时自动创建，含 seed 数据
- 前端通过 Vite proxy 将 `/api` 请求转发到后端 3000 端口
- 样式写在 `client/index.html` 的 `<style>` 标签中，无第三方 UI 库
- 取消已有预约的排班不会被删除，确保已有预约不受影响
- 同一教练在同一时段只能被一个学员预约
- JWT 有效期为 7 天
